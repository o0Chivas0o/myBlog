版本控制器 (Version Control Systems)

  • 在未有工作工具流之前,使用的文件夹目录拷贝,效率低下,沟通成本高.

  • 集中式版本控制系统

    • SVN、VSS

  • 分布式版本控制系统

    • git


      由于因在工作中只用过git版本控制器,也就是分布式版本控制器.所以只总结git的使用.并且最近在极客时间上学习苏玲老师的<玩转git三剑客>的视频课程,一起学习,一起总结.

git

  1. 最优的存储能力
  2. 非凡的性能
  3. 开源的
  4. 很容易做备份
  5. 支持离线操作
  6. 很容易定制工作流程
